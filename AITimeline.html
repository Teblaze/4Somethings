NavMesh

The Navmesh, or Navigation mesh is a mesh applied to the surface of a map and is used to help the AI characters traverse. Almost all 3D video games require the Navmesh in order to allow the NPCs movement. Usually, characters are not scripted to follow a certain path (Unless they have been coded to), the Navmesh is a special type of code that lets the characters understand where they can walk. This is extremely useful to developers as they do not need to code what path the AI takes.
